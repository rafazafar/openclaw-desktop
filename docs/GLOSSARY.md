- **OpenClaw**: the upstream automation/agent gateway.
- **Gateway**: OpenClaw runtime process that handles messaging/integrations.
- **Desktop app**: the consumer UI for managing OpenClaw.
- **Manager (clawctl)**: local control plane service used by the desktop app.
- **Integration**: a connectable service (Gmail, Calendar, Telegram).
- **Capability**: user-facing ability (read email, send email).
- **Permission**: toggle/setting controlling a capability.
- **Policy**: enforcement rules limiting actions even if the agent is prompted.
- **Support bundle**: exported diagnostics zip, redacted.
